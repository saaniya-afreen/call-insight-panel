# MPA Call Logs

## Project Info

MPA Call Logs is a call logging and analytics application.

## Technologies Used

This project is built with:

- Vite
- TypeScript
- React
- shadcn-ui
- Tailwind CSS
- Supabase (Backend)

## Local Development

To run locally, you need Node.js & npm installed.

```sh
# Clone the repository
git clone <YOUR_GIT_URL>

# Navigate to the project directory
cd <YOUR_PROJECT_NAME>

# Install dependencies
npm install

# Start the development server
npm run dev
```

## Deployment

The application can be deployed and published for public access.
